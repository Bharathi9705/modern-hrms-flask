# modern-hrms-flask
A modern Human Resource Management System (HRMS) built with Flask, SQLite, HTML, CSS, and JavaScript featuring employee management, leave tracking, attendance management, dashboard analytics, dark mode, glassmorphism UI, CSV export, and responsive SaaS-inspired design.
# Modern HRMS System

Modern HRMS (Human Resource Management System) built using Flask, SQLite, HTML, CSS, and JavaScript.

## Features

- Employee Directory Management
- Leave Management System
- Attendance Tracking
- Dashboard Analytics
- Dark/Light Mode
- Glassmorphism UI
- Animated Analytics Cards
- CSV Export
- Search & Filters
- Loading Skeletons
- Toast Notifications
- Responsive Design

## Tech Stack

### Frontend
- HTML5
- CSS3
- JavaScript

### Backend
- Flask
- SQLite
